## 📋 Technical Details

### **Platform Requirements**
- **Minimum:** Android 6.0 (Marshmallow, API 23)
- **Target:** Android 16 (API 36)
- **Recommended:** Android 8.0+ for best experience

### **Permissions Required**
- **All Files Access (MANAGE_EXTERNAL_STORAGE)** - Required to access mod files across device storage
- **Read Media Images** - Import custom sprites and textures for mods
- **Read Media Video** - Attach tutorial videos to mod packages
- **Internet** - Crash reporting only (no ads or tracking)

### **File Format Support**
- **INI Files** (.ini, .txt)
- **Configuration Files** - Standard INI format with sections and key-value pairs
- **Comments** - Preserves single-line and inline comments
- **Unicode** - Full UTF-8 support

### **Storage**
- **App Size:** ~15 MB
- **Cache:** Minimal (temporary files only)
- **User Data:** Stored only in mod project folders you choose

## 🎊 Getting Started

**First Time Users:**

1. **Grant Permissions** - Allow file access when prompted
2. **Choose Theme** - Select Light, Dark, or Dynamic in settings
3. **Pick a Font** - Try different fonts to find your favorite
4. **Open a Mod** - Tap the folder icon to browse for .ini files
5. **Start Editing** - Tap sections to expand, tap values to edit
6. **Save Your Work** - Tap the save icon when done

**Tips:**
- Long-press properties to drag and reorder
- Toggle sections by tapping headers
- Use boolean switches for true/false values
- Double-tap back button to exit app
